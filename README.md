# Industrial Vision Toolkit

A collection of reusable tools and utilities for industrial machine vision applications.

## Features

- Camera utilities
- Image processing helpers
- ROI tools
- Calibration utilities
- OCR helpers
- Barcode helpers
- Geometry functions
